# ブラウザ TCG ゲーム

ブラウザ単体で動くターン制カードゲームです。インストール不要、URLを開くだけで遊べます。

🎮 **[今すぐ遊ぶ → https://ugu00x-cell.github.io/tcg-game/](https://ugu00x-cell.github.io/tcg-game/)**

## ゲーム概要

| 項目 | 内容 |
|------|------|
| プレイヤー | 1人 vs CPU |
| 山札 | 各40枚（10種類のモンスター） |
| 勝利条件 | 相手の戦場のモンスターをすべて倒す |
| 先攻 | ランダム決定 |

## ルール

1. **毎ターンの流れ**
   カードを1枚以上マナにセット → モンスターを召喚 → 攻撃フェーズ → ターン終了

2. **マナシステム**
   カードをマナゾーンにセットするとマナが増える。モンスターのコスト分のマナが必要。

3. **戦闘**
   攻撃力（ATK）が高い方が勝ち。同値は相打ち（両方墓地）。

4. **召喚酔い**
   召喚したターンは攻撃できない。

5. **1ターン目**
   先攻・後攻ともに攻撃不可。

## CPU の思考ロジック

- **マナセット**: 手札の最強カードを目標に、必要枚数だけ弱いカードをマナにセット
- **召喚**: マナの許す限り最強カードを繰り返し召喚
- **攻撃**: 自分より ATK が低い相手のみ攻撃（自滅を回避）。倒せる中で最強を優先
